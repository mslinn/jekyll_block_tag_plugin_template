--- description: Jekyll plugin template demonstration layout: default title: Demonstration ---

Given this markup in an HTML file: 

{% raw %}{% block_tag_template param1="Today is a wonderful day!" %}
Hello, world!
{% endblock_tag_template %}{% endraw %}

The rendered HTML from the block tag looks like this:

{% block_tag_template param1="Today is a wonderful day!" %} Hello, world! {% endblock_tag_template %}
