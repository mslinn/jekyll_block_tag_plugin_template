Tag

Two examples:

  1. Given this markup in an HTML file: 
    {% raw %}{% tag_template smile size='3em' %}{% endraw %}
    The generated HTML is: {% capture html %}{% tag_template smile size='3em' %}{% endcapture %} 
    {{ html | escape }}
    The rendered HTML from the tag looks like this: {% tag_template smile size='3em' %}
  2. Given this markup in an HTML file: 
    {% raw %}{% tag_template emoji_and_name list size='1em' %}{% endraw %}
    The generated HTML is: {% capture html %}{% tag_template emoji_and_name list size='1em' %}{% endcapture %} 
    {{ html | escape }}
    The rendered HTML from the tag looks like this: {% tag_template emoji_and_name list size='1em' %}

Block Tag

Given this markup in an HTML file: 

{% raw %}{% block_tag_template param1="Today is a wonderful day!" %}
Hello, world!
{% endblock_tag_template %}{% endraw %}

The rendered HTML from the block tag looks like this:

{% block_tag_template param1="Today is a wonderful day!" %} Hello, world! {% endblock_tag_template %}

With Attribution

Given this markup in an HTML file: 

{% raw %}{% block_tag_template attribution param1="Today is a wonderful day!" %}
  Hello, world!
{% endblock_tag_template %}{% endraw %}

The rendered HTML from the block tag looks like this:

{% block_tag_template attribution param1="Today is a wonderful day!" %} Hello, world! {% endblock_tag_template %}

Filter

Given this markup in an HTML file: 

{% raw %} Search for {{ "joy" | my_filter_template }}!{% endraw %}

The rendered HTML from the block tag looks like this:

Search for {{ "joy" | my_filter_template }}!
